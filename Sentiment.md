###### **AI Sentiment Analytics Platform**



An end-to-end AI-powered Sentiment Analysis Dashboard that analyzes textual data (tweets/reviews) to uncover customer sentiment, keyword trends, and actionable business insights.



###### **Project Overview**



This project uses Natural Language Processing (NLP) and Machine Learning to classify text into:



\* Positive

\* Neutral

\* Negative



It also provides:



\* Interactive dashboards

\* Keyword analysis

\* AI-generated insights

\* Sentiment trends



###### **Key Features**



**1. Sentiment Classification**



\* Predicts sentiment of user input in real-time

\* Supports text like tweets, reviews, or feedback



**2. Sentiment Distribution Dashboard**



\* Visual breakdown of sentiment categories

\* Helps understand overall user perception



**3. Keyword Analysis**



\* Extracts most frequent words from dataset

\* Highlights important discussion topics

\* Stopwords removed for meaningful insights



**4. AI Insights Generation**



Automatically generates business insights such as:



&#x20; \* Customer satisfaction trends

&#x20; \* Common complaints

&#x20; \* Improvement suggestions



**5. Interactive UI**



\* Built using Streamlit

\* Clean and modern dark-themed dashboard

\* Easy to use for non-technical users



Machine Learning Pipeline



###### **Data Preprocessing**

\* Stopword removal
